# ArhivskiFenomeni-NFTzbirkaPoznatiNepoznati
Na nekoliko različitih načina opisuno arhivsko gradivo zbirke Ulčnik - NFT zbirke Poznati nepoznati i njihovih dijelova kao različitih arhivskih fenomena. 
Gradivo se opisuje posredstvom dvije opisne norme, jedne norme oblikovanja i dvije serijalizacijske norme. Rezultati opisa su ISAD(G) EAD XML opis zbirke Ulčnik i njezinih dijelova te RiC RDF opisi digitaliziranih objekata iz Zbirke Ulčnik i NFT zbirke. 
U fokusu ovih opisa jesu različite materijalno-diskurzivne konfiguracije koje proizlaze iz gradiva iste osnovne provenijencije.
